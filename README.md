# Лендинг «Иванычи» — тестовое задание

Одностраничный лендинг для компании инженерных систем (вентиляция, кондиционирование, отопление).

## Стек

- HTML5, CSS3, ванильный JS
- PHP для обработки форм (отправка писем)
- Splide — слайдеры

## Структура

- `index.html` — разметка
- `css/style.css` — базовые стили
- `css/premium.css` — доработки под этот лендинг
- `js/main.js` — логика (модалки, калькулятор, табы, бургер)
- `send-mail.php` — приём форм, антиспам, отправка на почту (SMTP/Brevo/mail)

## Запуск

Разместить файлы на хостинге с PHP. Для отправки форм в `send-mail.php` указать почту получателя и настроить SMTP (пароль лучше задать переменной окружения `SMTP_PASS` на сервере).

Локально можно открыть `index.html` в браузере — вёрстка и скрипты работают; отправка форм потребует PHP-сервер.
